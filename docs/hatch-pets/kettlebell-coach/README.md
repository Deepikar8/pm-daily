# Kettlebell Coach Hatch Pet

Animated Codex pet package for the Product Gym coach mascot.

## Files

- `pet.json` - Codex pet manifest.
- `spritesheet.webp` - Final 8x9 animated pet atlas.
- `contact-sheet.png` - Visual QA sheet for all animation rows.
- `validation.json` - Atlas validation output.
- `review.json` - Frame extraction and component review output.

## Notes

- Display name: Kettlebell Coach
- Intended use: Product Gym mascot / trainer personality
- Validation status: passed with no atlas warnings or errors
- Preview video generation was skipped locally because the hatch-pet video helper needs a newer Python runtime than the available Pillow environment.
